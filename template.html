Example of how jQuery can simplify things

Pure Javascript to make all divs on the page change to red when clicked on

      var i, 
        divs = document.getElementsByTagName('div');
      for(i = 0; i < divs.length; i+=1) {
        divs[i].onclick = function() {
          this.style.backgroundColor = 'red';
        }
      }

jQuery to do the same things

      $('div').click(function() {
        $(this).css('background-color', 'red');
      });

How do validate a form?

Concepts covered

  • Event handling
  • DOM
  • Code structure

Events

  • Many different types: click, scroll, form submit
  • Can be triggered by the user or you can trigger your own
  • A good portion of Javascript coding is listening to and responding to events
  • With Javascript, you can control the flow of events. Beware: Can break the UI.

DOM

  • Structure of the page represented in code
  • Any part of the page can be accessed, modified, deleted in Javascript
  • You can also add objects to the page

Code structure

Code must follow certain rules or it will fail

  • Decisions (conditionals) - if/else
  • Repeat tasks (loops) - for/while
  • Reusable items (variables, functions, objects) - var, function(), {}

Form example

Here's our form:

<form id="loginForm" action="/login">
  <input type="text" name="username" placeholder="Username">
  <input type="password" name="password" placeholder="Password">
<button type="submit">submit</button>
</form>
<div id="messages"></div>

Using the DOM and events, we can use Javascript to listen to when the form is submitted (event), and check the values in the form (DOM).

Form example

Here's the Javascript:

$(function() {
    $('#myForm').on('submit', function() {
        if ($('#user').val() === '') {
            $('#messages').html('You must enter a username.');
            $('#user').addClass('error');
        
        } else if ($('#password').val() === '') {
            $('#messages').html('You must enter a password.');
            $('#password').addClass('error');
        } else {
            $('#messages').html('Form is valid');
        }
        return false;
    });
});

We listen for a submit event and then use an if statement to inspect the DOM and see if we have values in the form. If not, we display a message to the user.
